# Vaastu Services, Inc - Frontend

A modern, responsive, and premium static frontend web application for Vaastu Services, Inc, a licensed General Contractor in West Palm Beach, FL.

## 🚀 Live Demo

**[View the Live Deployment on Vercel](https://vaastu-frontend.vercel.app)**

## 🛠️ Technologies Used
- HTML5
- CSS3 (Vanilla, custom properties, glassmorphism, responsive grid/flexbox)
- JavaScript (Vanilla, Intersection Observer for micro-animations)

## ✨ Features
- **Modern UI/UX**: Clean dark blue and white color palette matching the brand identity.
- **Responsive Design**: Fully mobile-responsive with a collapsible hamburger menu.
- **Scroll Animations**: Smooth fade-up animations triggered on scroll for a dynamic feel.
- **Optimized Assets**: Vector-based SVG logo rendering for maximum crispness at any resolution.

## 📂 Project Structure
- `index.html` - The main single-page application structure.
- `styles.css` - All styling, variables, animations, and media queries.
- `script.js` - Interactivity (mobile menu, scroll observer, sticky header).
- `assets/` - Contains the SVG logo and other media.

## ⚙️ How to Run Locally
1. Clone this repository.
2. Open the folder in VS Code (or your preferred editor).
3. Right-click `index.html` and select **Open with Live Server**.
4. Alternatively, just double-click `index.html` to open it directly in your browser.
